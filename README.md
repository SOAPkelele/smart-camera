# Умная камера

### Для запуска скрипта:

```
# скачивание репозитория со скриптом и весами
git clone https://github.com/SOAPkelele/smart-camera.git 

# перемещение в папку
cd smart-camera/app

# установка зависимостей
pip install - r requirements.txt

# Перед запуском нужно настроить config.py

# запуск модели
python detect.py
```
### Требования для запуска:
<b>Raspberry PI 4b b03114 и линзы.</b> 

Технические характеристики:
- Broadcom BCM2711 SoC
- Quad-core ARM Cortex-A72 1.5 GHz processor
- 2GB of LPDDR4 SDRAM
- Full-throughput Gigabit Ethernet
- Dual-band 802.11ac wireless networking
- Bluetooth 5.0
- OS – Raspberry Pi OS 11 Bullseye
- Разрешение линзы 1280 x 960
