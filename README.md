# Умная камера

###Для запуска скрипта:

```
# скачивание репозитория со скриптом и весами
git clone https://github.com/SOAPkelele/smart-camera.git 

# перемещение в папку
cd smart-camera/app

# установка зависимостей
pip install - r requirements.txt

# Перед запуском нужно настроить config.py

# запуск модели
python detect.py
```
